**Smart Course Companion - Academic Management Platform**

University students need better tools to track courses, assessments, deadlines, and academic progress. While LMS platforms provide course materials, they lack personalized tracking, progress visualization, and planning features.

This project involves designing and developing a web-based Smart Course Companion that enables students to:
1. Organize courses and manage assessments
2. Track grades and deadlines
3. Visualize academic progress

The system should also provide instructors/administrators with basic tools to manage course structures and assessments.

**Core Challenge**: Bridge the gap between generic LMS functionality and personalized student academic planning needs.
